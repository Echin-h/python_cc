### python 的输出
```python
a =3
print('hello world'，a,"hello world")
```
"" 与 ''没有区别
a可以直接存放在两个字符串之间
不需要一些main主函数，写分号之类的繁琐操作


### python 的输入
```python 
num = int(input())
```
input() 输入的是字符串类型的
int()类型转换
num不需要跟类型
突出一个简单 

### python 的 fstring
```python
a, b = input().split(",")  
a = int(a)  
b = int(b)  
product = a * b  
print(f"the product of {a} and {b} is: {product}")
```
input().split(',') 可以直接以（，）的形式输入
print()部分  用fstring方式输出， {}中要放入变量 -> 突出一个想放啥放啥
